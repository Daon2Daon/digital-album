# 🖼️ 전체 화면 표시 가이드

## ✅ 완료!

iPad mini 1세대에서 이미지를 **전체 화면**으로 표시하는 기능이 추가되었습니다!

---

## 🎯 표시 모드

### 1️⃣ 전체 화면 채우기 (Cover) - **기본값, 권장**

```
┌─────────────────────┐
│   [  I M A G E  ]   │  ← 화면을 완전히 채움
│   [  F U L L    ]   │
│   [  S C R E E N]   │
└─────────────────────┘
```

**특징**:
- ✅ 화면을 완전히 채워서 **몰입감** 최고
- ✅ 검은 여백 없음
- ⚠️ 이미지 일부가 잘릴 수 있음 (비율이 다른 경우)

**추천 용도**:
- 풍경 사진
- 추상적인 이미지
- 분위기 있는 슬라이드쇼

---

### 2️⃣ 이미지 전체 보기 (Contain)

```
┌─────────────────────┐
│████████████████████│  ← 검은 여백
│   [ I M A G E ]     │
│████████████████████│  ← 검은 여백
└─────────────────────┘
```

**특징**:
- ✅ 이미지 **전체**가 보임
- ✅ 잘림 없음
- ⚠️ 화면 주변에 검은 여백 생김 (비율이 다른 경우)

**추천 용도**:
- 인물 사진
- 작품/포트폴리오
- 텍스트가 있는 이미지

---

## 🔧 설정 방법

### 1단계: 관리자 페이지 접속

```
http://localhost:8754/admin
```

### 2단계: 슬라이드쇼 설정 섹션으로 스크롤

하단의 **"슬라이드쇼 설정"** 섹션을 찾으세요.

### 3단계: 표시 모드 선택

**"표시 모드"** 드롭다운에서 선택:
- **전체 화면 채우기 (권장)** ← 기본값
- **이미지 전체 보기 (비율 유지)**

### 4단계: 설정 저장

**"설정 저장"** 버튼 클릭

### 5단계: iPad에서 확인

1. **자동 반영**: 최대 10분 대기
2. **수동 반영**: Safari 새로고침 (⌘+R)
3. **즉시 반영**: 페이지 다시 열기

## 📱 iPad mini 1세대 최적화

### 화면 정보
- **해상도**: 1024 × 768
- **비율**: 4:3

### 권장 이미지 크기
- **최적**: 1024 × 768 (4:3 비율)
- **대안**: 1600 × 1200, 2048 × 1536
- **파일 크기**: 500KB 이하 권장

---

## 🎨 이미지 준비 가이드

### 전체 화면 채우기 모드용

1. **4:3 비율로 편집**
   - Photoshop/GIMP에서 1024 × 768로 자르기
   - 또는 자동 리사이즈 (업로드 시 1920px로 자동 조정)

2. **중요한 부분을 중앙에**
   - 인물 얼굴, 주요 객체는 중앙 배치
   - 가장자리는 잘려도 괜찮은 부분

### 이미지 전체 보기 모드용

- **어떤 비율이든 OK**
- 세로 이미지, 가로 이미지 모두 완전히 보임
- 파일 크기만 주의 (메모리 512MB)

## 🧪 테스트 방법

### PC에서 미리보기

1. Chrome 열기
2. `F12` → 개발자 도구
3. `Ctrl+Shift+M` → 디바이스 툴바
4. "iPad Mini" 선택
5. `http://localhost:8754/` 접속
6. 두 모드 비교

### iPad에서 테스트

1. Safari에서 `http://[NAS_IP]:8754/` 접속
2. 첫 이미지 확인
3. 전체 화면으로 보이는지 확인
4. 관리자 페이지에서 모드 변경
5. Safari 새로고침
6. 차이 확인

## 🐛 문제 해결

### Q: 이미지가 잘려서 보여요
**A**: "이미지 전체 보기" 모드로 변경하세요.

### Q: 화면에 검은 여백이 있어요
**A**: "전체 화면 채우기" 모드로 변경하거나, 이미지를 4:3 비율로 편집하세요.

### Q: 설정 변경이 반영 안 돼요
**A**: 
1. iPad Safari 강제 새로고침 (`⌘+Shift+R`)
2. 또는 10분 대기
3. 또는 페이지 다시 열기

### Q: 어떤 모드가 더 나은가요?
**A**: 
- **몰입감 중시** → 전체 화면 채우기
- **정확성 중시** → 이미지 전체 보기

## 💡 고급 팁

### 두 모드 혼합 사용

## 📚 기술 정보

### 구현 방식
- CSS `background-size` 속성 사용
- iOS 9.3.6 완벽 지원 ✅
- JavaScript로 동적 적용

### 브라우저 호환성
| 기능 | iOS 9.3.6 | 설명 |
|------|-----------|------|
| `background-size: cover` | ✅ | 전체 화면 |
| `background-size: contain` | ✅ | 비율 유지 |
| `background-position: center` | ✅ | 중앙 정렬 |
