# 🖥️ 완전 전체 화면 설정 가이드

## 📱 iPad에서 주소창 완전히 숨기기

iPad mini 1세대에서 **주소창, 탭바, 툴바를 모두 숨기고** 완전한 전체 화면으로 디지털 액자를 실행할 수 있습니다!

---

## ✅ 설정 방법 (3단계)

### 1단계: Safari로 뷰어 접속

iPad mini 1세대의 Safari에서:

```
http://[NAS_IP]:8754/
```

슬라이드쇼가 정상 작동하는지 확인합니다.

---

### 2단계: 홈 화면에 추가

Safari에서:

1. **공유 버튼** 탭 (상단 도구바의 ↗️ 아이콘)
2. **"홈 화면에 추가"** 선택
3. 이름 확인: **"디지털 액자"**
4. **"추가"** 버튼 탭

<details>
<summary>📸 스크린샷 가이드</summary>

```
┌─────────────────────┐
│  Safari 화면        │
│                     │
│  ↗️ 공유 버튼 탭    │
│      ↓              │
│  [홈 화면에 추가]   │
│      ↓              │
│  이름: 디지털 액자  │
│      ↓              │
│  [추가] 버튼 탭     │
└─────────────────────┘
```

</details>

---

### 3단계: 홈 화면 아이콘으로 실행

1. 홈 화면에 생성된 **"디지털 액자"** 아이콘 탭
2. 전체 화면으로 실행됩니다!

**결과**:
- ✅ 주소창 없음
- ✅ 탭바 없음
- ✅ 도구바 없음
- ✅ 완전한 전체 화면
- ✅ 오직 이미지만!

## 💡 추가 최적화

### iPad 자동 잠금 해제

전체 화면으로 실행 후:

```
설정 > 디스플레이 및 밝기 > 자동 잠금 > 안 함
```

### Guided Access (앱 고정)

실수로 종료되지 않도록:

1. **설정 활성화**:
   ```
   설정 > 일반 > 손쉬운 사용 > Guided Access > 켜기
   ```

2. **사용**:
   - 디지털 액자 실행 후
   - **홈 버튼 3번 빠르게 누르기**
   - 화면 잠금 영역 설정
   - **시작** 탭

3. **해제**:
   - 홈 버튼 3번 빠르게 누르기
   - 암호 입력

## 🔧 아이콘 커스터마이징 (선택사항)

기본 아이콘 대신 원하는 이미지를 아이콘으로 사용할 수 있습니다.

### 방법 1: 간단한 텍스트 아이콘 (현재)

현재는 기본 스크린샷이 아이콘으로 사용됩니다.

### 방법 2: 커스텀 아이콘 만들기

1. **아이콘 이미지 준비**:
   - 크기: 192×192 픽셀
   - 형식: PNG
   - 디자인: 액자 모양, 사진 아이콘 등

2. **서버에 업로드**:
   ```bash
   # public 폴더에 아이콘 파일 복사
   cp icon-192.png /path/to/digital-album/public/
   cp icon-152.png /path/to/digital-album/public/
   ```

3. **홈 화면에 다시 추가**

### 아이콘 생성 도구

- **온라인**: 
  - [favicon.io](https://favicon.io/)
  - [realfavicongenerator.net](https://realfavicongenerator.net/)

- **Photoshop/GIMP**:
  - 192×192 캔버스
  - 검은 배경
  - 하얀색 액자 아이콘
  - PNG로 저장

## 🚀 자동 시작 설정 (고급)

iPad를 켜면 자동으로 디지털 액자가 실행되도록 설정:

### iOS 9.3.6 제약사항

- 완전 자동 시작 불가능
- Guided Access로 화면 고정 권장

### 권장 워크플로우

1. iPad 전원 켜기
2. 잠금 해제
3. 홈 화면에서 "디지털 액자" 아이콘 탭
4. Guided Access로 화면 고정 (선택)

## 🔍 문제 해결

### Q: 홈 화면에 추가했는데 주소창이 보여요

**A**: 
1. 홈 화면 아이콘을 **다시 추가**해보세요
2. 또는 Safari **설정 초기화**

### Q: 홈 화면 아이콘이 안 보여요

**A**: 
1. Safari에서 정상 실행되는지 확인
2. 공유 버튼(↗️)이 활성화되어 있는지 확인
3. iOS 버전 확인 (9.3.6 이상)

### Q: 실행하면 Safari로 열려요

**A**: 
- 홈 화면 아이콘을 **탭**하세요
- Safari 북마크가 아닌 **홈 화면 아이콘** 사용

### Q: 화면이 자꾸 꺼져요

**A**: 
```
설정 > 디스플레이 및 밝기 > 자동 잠금 > 안 함
```

### Q: 실수로 종료돼요

**A**: 
Guided Access 활성화:
```
설정 > 일반 > 손쉬운 사용 > Guided Access
```
