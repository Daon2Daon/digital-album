## 🏠 2단계: 홈 화면에 추가

### 방법

1. Safari 상단의 **공유 버튼(↗️)** 탭
2. **"홈 화면에 추가"** 선택
3. 이름: **"디지털 액자"** (기본값)
4. **"추가"** 탭

### 시각적 가이드

```
Safari 화면
    ↓
[↗️] 공유 버튼 탭
    ↓
메뉴에서
"홈 화면에 추가" 선택
    ↓
이름 확인
"디지털 액자"
    ↓
[추가] 버튼 탭
    ↓
✅ 홈 화면에 아이콘 생성!
```

## 💡 추가 팁

### 1. 자동 잠금 해제

디지털 액자가 꺼지지 않도록:

```
설정 > 디스플레이 및 밝기 > 자동 잠금 > 안 함
```

### 2. 화면 고정 (Guided Access)

실수로 종료되지 않도록:

```
설정 > 일반 > 손쉬운 사용 > Guided Access > 켜기
```

**사용**:
- 디지털 액자 실행 후
- 홈 버튼 3번 빠르게 클릭
- 화면 영역 설정
- "시작" 탭

**해제**:
- 홈 버튼 3번 빠르게 클릭
- 암호 입력

### 3. 화면 밝기 조정

```
설정 > 디스플레이 및 밝기 > 밝기 조정
```

또는:
- 자동 밝기 켜기 (환경에 맞춰 자동 조정)

## 🎨 아이콘 커스터마이징 (선택)

기본 아이콘 대신 예쁜 아이콘을 만들 수 있습니다!

### 방법 1: 아이콘 생성기 사용

PC 브라우저에서:

```
http://localhost:8754/icon-generator.html
```

1. 텍스트, 색상, 스타일 선택
2. "아이콘 생성" 클릭
3. 각 사이즈별로 다운로드
4. 파일 이름 변경:
   - `icon-152.png`
   - `icon-192.png`
5. 서버 `public` 폴더에 복사
6. iPad에서 홈 화면 아이콘 다시 추가

### 방법 2: 직접 만들기

Photoshop, GIMP 등으로:

- **크기**: 192×192 픽셀
- **배경**: 검은색 또는 원하는 색
- **내용**: 액자 아이콘, 사진, 이모지 등
- **저장**: PNG 형식

## 🔍 문제 해결

### Q: 홈 화면에 추가했는데 주소창이 보여요

**원인**: Safari 캐시 문제

**해결**:
1. 홈 화면 아이콘 **길게 눌러 삭제**
2. Safari에서 다시 접속
3. "홈 화면에 추가" **다시** 진행

### Q: "홈 화면에 추가" 메뉴가 안 보여요

**원인**: 
- 잘못된 URL
- 네트워크 문제

**해결**:
1. URL 확인: `http://[IP]:8754/`
2. 페이지가 완전히 로드될 때까지 대기
3. 공유 버튼(↗️) 다시 확인

### Q: 아이콘을 누르면 Safari로 열려요

**원인**: 북마크를 만든 경우

**해결**:
- **홈 화면 아이콘** 사용 (Safari 앱이 아님)
- 아이콘이 없으면 다시 추가

### Q: 실행하면 빈 화면이에요

**원인**: 서버 연결 문제

**해결**:
1. 서버 실행 확인: `node server.js`
2. 네트워크 연결 확인
3. iPad Wi-Fi 확인
