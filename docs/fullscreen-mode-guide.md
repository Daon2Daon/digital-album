# 전체 화면 모드 가이드

## 🎯 개요

iPad mini 1세대에서 이미지를 전체 화면으로 표시하는 기능이 추가되었습니다.

두 가지 표시 모드를 선택할 수 있습니다:
- **전체 화면 채우기** (기본값): 이미지가 화면을 꽉 채웁니다
- **이미지 전체 보기**: 이미지 비율을 유지하며 전체가 보입니다

---

## 🔧 설정 방법

### 1. 관리자 페이지 접속

```
http://localhost:3000/admin
```

### 2. 슬라이드쇼 설정 섹션에서 "표시 모드" 선택

- **전체 화면 채우기 (권장)**: 이미지가 화면을 완전히 채웁니다
  - 장점: 화면을 꽉 채워 몰입도가 높음
  - 단점: 이미지 일부가 잘릴 수 있음
  
- **이미지 전체 보기 (비율 유지)**: 이미지가 완전히 보이도록 표시
  - 장점: 이미지 전체가 보임
  - 단점: 화면 주변에 검은 여백이 생김

### 3. "설정 저장" 클릭

---

## 📐 표시 모드 비교

### 전체 화면 채우기 (Cover)

```
┌───────────────────┐
│  I M A G E        │
│     F U L L       │  ← 화면 전체를 이미지가 채움
│        S I Z E    │
└───────────────────┘
```

**특징**:
- 화면을 완전히 채움
- 이미지 비율이 화면 비율과 다르면 일부가 잘림
- 세로 이미지는 좌우가 잘림
- 가로 이미지는 상하가 잘림

**권장 대상**:
- 풍경 사진
- 추상적인 이미지
- 디테일보다 분위기가 중요한 이미지

---

### 이미지 전체 보기 (Contain)

```
┌───────────────────┐
│███████████████████│  ← 검은 여백
│   I M A G E       │
│████████████████████│  ← 검은 여백
└───────────────────┘
```

**특징**:
- 이미지 전체가 보임
- 화면 비율과 다르면 검은 여백 생김
- 세로 이미지는 좌우에 여백
- 가로 이미지는 상하에 여백

**권장 대상**:
- 인물 사진
- 텍스트가 있는 이미지
- 모든 디테일을 봐야 하는 이미지

---

## 🎨 이미지 준비 팁

### iPad mini 1세대 화면 비율

- **해상도**: 1024 × 768 (4:3 비율)
- **권장 이미지 크기**: 1024 × 768 또는 그 배수

### 전체 화면 채우기 모드용

1. **4:3 비율로 편집**
   - 1024 × 768
   - 1600 × 1200
   - 2048 × 1536

2. **또는 중요한 부분을 중앙에 배치**
   - 잘려도 괜찮은 부분은 가장자리에
   - 중요한 요소는 중앙에

### 이미지 전체 보기 모드용

- 어떤 비율이든 OK
- 세로 이미지, 가로 이미지 모두 전체가 보임
- 파일 크기만 주의 (500KB 이하 권장)

3. **즉시 반영**: 페이지 다시 열기

## 🧪 테스트 방법

### PC에서 미리보기

1. Chrome 개발자 도구 (F12)
2. 디바이스 툴바 토글 (Ctrl+Shift+M)
3. "iPad Mini" 선택
4. 두 모드 비교

### iPad에서 테스트

1. Safari에서 뷰어 열기
2. 전체 화면 모드로 보기
3. 이미지가 어떻게 보이는지 확인
4. 관리자 페이지에서 모드 변경
5. Safari 새로고침 (또는 10분 대기)

## 🔧 고급 설정 (개발자용)

### CSS 수준 커스터마이징

`viewer.html` 파일의 CSS 수정:

```css
.slide {
    background-size: cover; /* 또는 contain */
    background-position: center center; /* 중앙 정렬 */
}
```

### 다른 배경 위치 옵션

```css
background-position: top; /* 상단 정렬 */
background-position: bottom; /* 하단 정렬 */
background-position: left; /* 좌측 정렬 */
background-position: right; /* 우측 정렬 */
```

## 🐛 문제 해결

### 이미지가 잘려서 보임

→ "이미지 전체 보기" 모드로 변경

### 화면에 검은 여백이 생김

→ "전체 화면 채우기" 모드로 변경
→ 또는 이미지를 4:3 비율로 편집

### 설정이 반영 안 됨

1. 관리자 페이지에서 설정 확인
2. iPad Safari 강제 새로고침 (Cmd+Shift+R)
3. 서버 재시작
4. 브라우저 캐시 삭제

---

## 📊 기술 정보

### 구현 방식

- **CSS `background-size`** 속성 사용
- iOS 9.3.6 완벽 지원
- JavaScript로 동적 적용

### 브라우저 호환성

| 속성 | iOS 9.3.6 | 설명 |
|------|-----------|------|
| `background-size: cover` | ✅ | 전체 화면 채우기 |
| `background-size: contain` | ✅ | 이미지 전체 보기 |
| `background-position` | ✅ | 이미지 위치 조정 |

## 📝 API 정보

### Settings 객체

```json
{
  "slideDuration": 10000,
  "transitionEffect": "fade",
  "transitionSpeed": 1000,
  "displayMode": "cover"
}
```

### displayMode 값

- `"cover"`: 전체 화면 채우기
- `"contain"`: 이미지 전체 보기
